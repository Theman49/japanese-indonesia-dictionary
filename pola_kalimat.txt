----------------------

+++++++++++++++++++++++ Bab 6 +++++++++++++++++++++++
======= Menuju ke suatu tempat ==========
----------------------------------------------
| S + は + TEMPAT + へ + 行きます（いきます）/ 来ます（きます）/ 帰ります（かえります） |
----------------------------------------------
-> アリさんは日本へ来ました。




======= Menggunakan alat untuk melakukan kegiatan ==========
----------------------------------------------
| S + は + KB(Kendaraan/alat) + で + 行きます（いきます）/ 来ます（きます）/ 帰ります（かえります） |
----------------------------------------------
partikel で digunakan untuk perantara suatu cara dan sarana, dalam hal ini sarana transportasi.
-> 私は電車（でんしゃ）で行きます。
-> 山田さんはタクシーでホテルへ来ます。
Jika pergi dengan jalan kaki, digunakan 歩いて（あるいて）. Tidak perlu ditambahkan partikel で。
-> 駅（えき）から歩いて（あるいて）帰りました（かえりました）。




======= Melakukan sesuatu bersama orang lain ==========
----------------------------------------------------
| S + は + KB(orang/hewan) + と + KK + ます/ました |
----------------------------------------------------
partikel と menunjukkan kebersamaan

-> 私は家族（かぞく）と日本へ来ました。

Jika melakukan sesuatu sendirian, digunakan 一人で（ひとりで）dan tidak perlu ditambahkan partikel と.

-> 一人でジャカルタへ行きます。





======= Saat pergi ke tempat lain ==========
----------------------------------------------------
| S + は + WAKTU + に + TEMPAT + へ + KK |
----------------------------------------------------
に -> keterangan waktu
へ -> keterangan tempat tujuan

-> 私は10時にアメリカへ行きます。

いつ digunakan untuk menanyakan waktu (bulan, tanggal, hari, jam) namun tidak perlu ditambahkan partikel に。

-> いつ日本へ来ましたか。


---------------------------------------------------------
^^^^^^ Lampiran : Akhiran ～よ　^^^^^^
---------------------------------------------------------
digunakan untuk menyampaikan tanggapan atau pendapat si pembicara kepada lawan bicaranya yang belum diketahuinya.

-> 北海道（ほっかいどう）に馬（うま）がたくさんいますよ。




======= Apabila ingin menyangkal secara keseluruhan dari pertanyaan ==========

| KB1 + は + 何 + [へ]も + KK Negatif / KK Negatif lampau |
-----------------------------------------------------------
-> 今朝（けさ）何を食べましたか。
	何も食べませんでした。

======= Saya Mengerti ... / Saya Tidak Mengerti ...  ==========
-----------------------------------------------------
| S + は + KB + が + わかります	  + です    |
| S + は + KB + が + わかりません + です    |
-----------------------------------------------------

kata あまり(tidak begitu) atau 全然（ぜんぜん）(sama sekali) bisa diberikan di depan わかりまえん untuk penekanan.

-> アリさんは日本語（にほんご）がわかります。
-> 私は漢字（かんじ）が全然（ぜんぜん）わかりません。

+++++++++++++++++++++++ Bab 15  +++++++++++++++++++++++
======= Kelompok Kata Kerja ==========
---------------
KK Kelompok 1 |
---------------
Jika bunyi terakhir bentuk ますberakhir dengan bunyi pada kolom い.
contoh : 立ちます（たちます）、飲みます（のみます）。

---------------
KK Kelompok 2 |
---------------
Hampir semua KKtermasuk kelompok ini, bunyi terakhir bentuk ます berakhir dengan bunyi pada kolom え, namun ada juga sebagian pada kolom い.
contoh : 食べます（たべます）、見せます（みせます）、見ます（みます）, dst.


---------------
KK Kelompok 3 |
---------------
yang termasuk dalam kelompok ini yaitu します、KB aksi + します、dan 来ます（きます）.


--------------- Bentuk て / て形 ---------------
-----------
KK Grup 1 |
-----------
========================================
ます形		|	て形
========================================
か-きます	|	か-いて
い-きます	|	か-いて*
いそ-ぎます	|	いそ-いで
の-みます	|	の-んで
よ-びます	|	よ-んで
かえ-ります	|	かえ-って
か-います	|	か-って
ま-ちます	|	ま-って
か-します	|	か-して



-----------
KK Grup 2 |
-----------
========================================
ます形		|	て形
========================================
たべ-ます	|	たべ-て
